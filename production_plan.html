작성: 2026-06-11 · 기반: C:\GPT\seongnam-after-school-map (Codex 구버전) 재활용

목표

성남시 모든 초등학교(74개교)의 채용 공고를 매일 자동으로 점검하고, 늘봄·방과후학교 강사·디지털튜터 관련 새 공고가 뜨면 텔레그램과 메일로 즉시 알림. 지도에서 구 → 동 → 학교 순서로 탐색하는 위치 기반 인터페이스 제공.

⚙️ 남은 설정 (사용자 직접 — 약 10분)

1. 텔레그램 봇 만들기

  1. 텔레그램에서 @BotFather 검색 → /newbot → 이름 입력 → 봇 토큰 복사
  2. 만든 봇과 대화 시작 (/start 한 번 보내기)
  3. 브라우저에서 https://api.telegram.org/bot<토큰>/getUpdates 열기 → "chat":{"id":123456789 의 숫자가 chatId

2. Gmail 앱 비밀번호

  1. Google 계정 → 보안 → 2단계 인증 켜기
  2. 앱 비밀번호 생성 → 16자리 복사

3. 설정 파일 작성

cd C:\claude\projects\생존앱_성남채용알림_20260611\config
copy notify_config.example.json notify_config.json

서울시 버전 (2026-06-12 추가)

  • 웹: /seoul/ 경로 — 상단 성남↔서울 전환 버튼. 같은 app.js를 지역 설정(REGION_CONFIG)으로 재사용
  • 학교 데이터 (data/seoul_schools.json): OSM Overpass(이름·좌표) + NEIS 학교명 개별조회(주소·전화·홈페이지) + Nominatim 역지오코딩(동). 재구축: node scripts/fetch_seoul_schools_v2.mjs
  • ⚠️ NEIS는 키 없이 호출당 5행 제한 + accept 헤더 보내면 500 오류 — 학교명 개별조회로 우회
  • 채용 크롤러 (monitor_seoul_jobs.mjs): 서울 11개 교육지원청 구인 게시판({청}.sen.go.kr/FUS/JO/JOL11.do) — 초등+키워드 필터, 마감 지난 공고 제외. 청 코드: dbedu 동부 / sbedu 서부 / nbedu 남부 / bbedu 북부 / jbedu 중부 / gdspedu 강동송파 / gsycedu 강서양천 / gnscedu 강남서초 / dgedu 동작관악 / sdgjedu 성동광진 / sbgbedu 성북강북
  • 서울은 지원청 게시판이 중앙화되어 있어 학교 홈페이지 개별 크롤링은 하지 않음. work.sen.go.kr(일자리포털)은 JS 앱이라 직접 크롤링 불가
  • 알림: notify.py가 성남+서울+양성교육을 합쳐 발송
